# arknights-tools/story-tl

Generates raw locale files from extracted codes

## Install
#### Requirements
* NodeJS 8+ / npm 5+

#### Install packages
```
npm install
```

## Prepare input data
#### Story Files
* Add story files (`.txt`) into `/input`

## Run
* Run by executing command line
```
npm run start"
```
* See output in `/output/<title>.txt`
